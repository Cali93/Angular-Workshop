Première étape Mettez-vous par équipe de 2 et créez une branche. Créer une branche sur le repo :
  1. git clone https://github.com/Cali93/Angular-Workshop.git
  2. cd Angular-Workshop
  3. VOIR ETAPE DE CONFIGURATION CI DESSOUS
  4. git branch mybranch
  5. git checkout mybranch
Deuxième étape Configurez et initialisez l'application. Exécutez les commandes suivantes dans le dossier de ce projet:
  1. npm install
  2. npm start
  3. Rendez-vous sur votre localhost:4200
Troisième étape Créez votre premier module et component. Exécutez les commandes suivantes dans le dossier de ce projet:
  1. Nous allons d'abord générer un module : ng generate module NomDeVotreModule --routing
  2. Ensuite nous allons générer un component "parent" qui sera la page qui regroupera tout les compenents du module. ng generate component NomDeVotrePage -m NomDeVotreModule
  3. Il est désormais possible de créer d'autres components enfants à la page ng generate component header -m NomDeVotreModule
  4. Vous pouvez désormais coder vos différentes fonctionnalités.
 Quatrième étape Codez les différentes fonctionnalités. Dans le dossier de votre module :
  1. Rendez-vous sur le dossier du component enfant (le header par exemple) et cutomisez-le
  2. Ce compenent devra contenir:
    1. SCSS

      Au moins une dizaine de lignes

    2. HTML

      Avec au moins :

      • Une présentation de votre groupe
      • Un component material design
      • Un bouton avec un event lié à une fonction dans le fichier .ts
      • Un système de templatating {{ monElement }}
    3. TypeScript

      C'est ici que vous écrirez votre business logic

      1. Pour votre élément dynamique :
        1. Créez d'abord le nom de votre variable et assignez lui un type (dans la classe ) monElement: string;
        2. Dans le constructeur de votre component, assignez une valeur à la variable comme suit : {{ fakeConstruct }}
        3. Dans votre .html interpollez votre variable entre braçelets{{ monElement }}
      2. Pour votre event binding :
        1. Créez d'abord le nom de boutton dans votre html {{monBtn}}
        2. Vous pouvez désormais créez votre fonction dans votre classe : {{fakeFunc}}
        3. Allez plus loin en customisant votre fonction
  3. Importez le component enfant dans l'html de la page parente grace au selector : {{ baliseCustom }}
  4. Vous pouvez désormais coder vos différentes fonctionnalités.
 Etape Bonus Allez plus loin Ajoutez un service, un pipe ou une directive à votre page

  1. npm build:prod
  2. Déployez ce dossier sur la plateforme de votre choix
